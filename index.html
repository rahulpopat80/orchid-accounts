Excel Operations Guide

Manage your entire accounts using Microsoft Excel sheets. You can backup your data or upload massive transactions at once.

1. Download Full Backup or Template

Download a fully formatted Excel workbook containing three sheets: one for your Account Heads (Chart of Accounts), one for Ledger Transactions, and one for Member Deposits. This functions as a complete backup that can be re-uploaded to restore your data in the exact same state in one click.

2. Formatting for Upload

You can upload a single Excel sheet with your transaction details. The sheet must contain the following columns:

  • Date (dd-mm-yyyy) - Transaction date
  • Type - Income, Expense, or Contra
  • Head ID or Category - Account head name or code
  • Amount - Value of the transaction
  • Mode - Cash or bank name (e.g. CBI BANK, Cash in Hand)
  • Reference - Receipt/Chq no (Optional)
  • Description - Narration (Optional)

Note: Missing account heads mentioned in the Category column will be automatically created in the system.
